Question 4 - Lasso Regression¶

Question Is there a need for Lasso regularization? Explain. (Compare test and training results)

Yes, there is a need for Lasso regularization. The linear regression model without regularization exhibits overfitting, as seen in the significantly lower training errors compared to test errors. This indicates that the model fits the training data too closely but struggles to generalize to new data, resulting in higher errors on unseen data.

Lasso regularization can help by adding a penalty term that encourages feature selection and reduces overfitting. It improves the model's ability to generalize to unseen data and selects the most important features leading to a more effective and robust model.

# Finding the best alpha

alphas=[np.exp(i) for i in range(-9, 3)]
k = 5
n = len(y)

# Shuffle the data
indices = np.arange(n)
np.random.shuffle(indices)
y=y.values.reshape(-1)
X_cv = X.iloc[indices]
y_cv = y[indices]
avg_rmse = []
for alpha in alphas:
    rmse_list = []
    
    for fold in range(k):
        # Calculate the size of each fold
        fold_size = n // k
        
        # Define the start and end indices of the test set
        start, end = fold * fold_size, (fold + 1) * fold_size
        
        test_indices = indices[start:end]
        train_indices = np.concatenate((indices[:start], indices[end:]))
        
        X_train, X_test = X_cv.iloc[train_indices], X_cv.iloc[test_indices]
        y_train, y_test = y_cv[train_indices], y_cv[test_indices]
        
        # Use Lasso for L1 regularization
        lasso_model = Lasso(alpha=alpha, max_iter=10000)
        lasso_model.fit(X_train, y_train)
        
        y_pred = lasso_model.predict(X_test)
        rmse = np.sqrt(mean_squared_error(y_test, y_pred))
        
        rmse_list.append(rmse)
        
    avg_rmse.append(np.mean(rmse_list))
    
    # Find the best alpha
best_alpha = alphas[np.argmin(avg_rmse)]
print(f"Best alpha:{best_alpha}")

# Define functions to calculate AIC, AICc, and BIC
def aic(n,rss,k):
    aic = n*np.log(rss/n)+2*k # For linear regression
    return aic 

def aicc(n,rss,k):
    aic = n*np.log(rss/n)+2*k
    aicc = aic + (2*k*(k+1)) / (n-k-1) # For linear regression
    return aicc

def bic(n,rss,k):
    bic = n*np.log(rss/n)+k*np.log(n) # For linear regression
    return bic
